História dos computadores.

(Voltar)

Tudo começou para facilitar o dia a dia, calculos matemáticos mais rápido e mais fácil. Ao longo do tempo inventaram calculadoras como o ábaco, régua de cálculo e a máquina de Pascal, depois vieram outras máquinas como a máquina de diferenças, engenho analítico (Isso tudo era sistema mecânico) e foi criado também a teoria de Boole (sistema lógico), até que chegou à segunda guerra mundial, que foi o grande incentivo para o desenvolvimento de computadores.

Os primeiros computadores foram o MARK 1, Colossus e o Eniak, criados para tarefas de desencriptação de mensagens inimigas e criação de armas mais inteligentes.

Sendo o Eniak o primeiro computador sem a necessidade de movimentar peças de forma manual. Porém seu tamanho era de um prédio, enorme!

Depois disso vieram os computares pessoais da IBM e da Apple e a criação da Microsotf.

Os avanços não param até hoje, e os computadores já são bem menores (hardware) e com capacidade de executar inúmeras tarefas ao mesmo tempo (software).

Como surgiu a internet?

(Voltar)

Criada nos EUA durante a guerra fria (1969) para manter contato entre seus militares e os cientistas, foi chamada de Arpanet. Depois passou a ser usada para pesquisas acadêmicas, e trocou o nome para internet.

Primeira conexão foi feita através da Arpanet em entre duas universidades e foi enviado a palavra LOGIN, mas só chegou à palavra LO.

O que são Clients?

(Voltar)

Client é o cliente, o produto/ dispositivos. É quem sempre inicia a comunicação com o servidor. Ex: Através do navegador web.

Quem faz essa interface é o desenvolvedor Front end. Exemplos de ferramentas para criar: HTML, CSS, Javascript.

O que são Servers?

(Voltar)

Server é o Servidor. Pode ser via web/internet ou local/físico.

É quem recebe uma requisição e devolve uma resposta para o cliente. Para isso exige um bom processamento.

O servidor pode ser um ou mais computadores (datacenters) com um único endereço. Nestes computadores podem ser guardados documentos, imagens, áudio, vídeo, etc. Exemplo: É onde hospedamos um site.

Tipos de servidores:

  • De arquivos: Armazena e compartilha informações entre equipamentos e usuários.Client - Internet - Server.
  • De segurança: É um intermediário.

    Firewall – Cria uma barreira de segurança para acessos externos.Client - FIREWALL - Internet - Server.

    Proxy – Servidor bloqueia sites que a empresa não permite.Client - Proxy - Internet - Server.

  • De streaming: Armazena mídia digital e a disponibiliza em uma rede.
  • De e-mail/ Web/ Database: Usam um banco de dados. Ex: Nuvem.
  • Intranet: Página de web de rede local, sem usar a internet.
  • DNS: Responsável por fazer com que o domínio seja localizado na internet. Usa o endereço IP para enviar um HTTP específico. (Tem que fazer o registro.br).

Comunicação entre Client x Server.

(Voltar)

A rapidez que um cliente recebe uma resposta do server, depende da velocidade da conexão. Precisa levar em conta a compatibilidade entre os navegadores (browsers) e levar em consideração diferentes resoluções na criação de websites - design de responsividade. Client - Internet (HTTP) - Server.

Linguagem de programação.

(Voltar)

É uma linguagem escrita com instruções e regras para criar um programa. A linguagem é um meio/ ferramenta de programação que ajuda a resolver um problema, mas não funciona sem a pessoa que programa, programador.

Ex: Javascript, Python, Java, PHP, C#...

Linguagem server-side:

São as linguagens usadas no lado do servidor. Ex: PHP, JS, JAVA, Ruby e C#.

Linguagem client-side:

São as linguagens usadas no lado do cliente. Ex: JS.

Para rodar o JS no Chrome precisa do Engire V8, responsável por executar códigos do JavaScript dentro do navegador Google Chrome, sem usar um intermediário.

O que é um documento HTML?

(Voltar)

HTML não é uma linguagem de programação! É um arquivo de marcação.

É um arquivo simples que contém texto formatado/ de marcação com tags HTML.

As tags são formadas por uma estrutura própria que precisa ter os sinais de “<” e “>" e precisa ser aberta e fechada com o sinal “/”. Ex:De abertura < p > e de fechamento < / p > .

Tipografia em HTML.

(Voltar)
  • < ! DOCTYPE html >: Informa que o documento é HTML e sua versão.
  • < html >: Define a raiz do documento em HTML.
  • < head >: Onde contém informações para o documento. Ex: Título do documento.
  • < meta >: As tags < meta > vão dentro do elemento < head > e são usadas para especificar o conjunto de caracteres, o autor do documento, as configurações da janela de visualização etc.
  • < title >: Define o título do documento.
  • < body >: É onde fica todo o conteúdo em que o usuário vê e interage.

  • < h1 > até o < h6 >: Define o tamanho do título.

  • < hr / >: Define uma linha entre os parágrafos.
  • < p >: É onde começa um novo parágrafo.
  • < Strong >: Para deixar a palavra em negrito.
  • < u >: Sublinhar uma palavra.
  • < i >: Deixar uma palavra em itálico.
  • < ul >: É uma lista não ordenada.
  • < ol >: É uma lista ordenada.
  • < li >: Define os itens de uma lista.
  • < a >: Colocar hyperlink no documento.
  • < a href= “#” >: É uma referência dentro de um documento a uma URL externa, seja um link ou um arquivo.
  • < mark >: É quando marcamos o texto de outra cor.
  • < small >: Diminuir o tamanho da letra.
  • < sup >: Define texto sobrescrito.
  • < sub >: Define o texto subscrito.
  • < blockquote >: Define uma citação.
  • < font >: Define fonte, cor e tamanho para o texto.
  • < del >: Define o texto que foi excluído de um documento.
  • < abbr >: Define uma abreviatura ou um acrónimo. Ex: HTML.

Como é uma estrutura básica de HTML?

(Voltar)

Exemplo:

< ! DOCTYPE html >

< html >

< head >

< title > Meu primeiro site. < / title >

< / head >

< hr / >

< body >

< h1 > Título. < / h1 >

< p > Parágrafo. < / p >

< / body >

< / html >
